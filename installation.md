# Installation & Deployment Guide

This guide covers setting up the development environment, running the app locally, and packaging it for Android.

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- npm or yarn package manager
- [Android Studio](https://developer.android.com/studio) (for building the APK/AAB)
- [Capacitor CLI](https://capacitorjs.com/docs/cli) (`npm install -g @capacitor/cli @capacitor/core`)
